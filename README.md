## ğŸ¯ Objectives of Work Package 2

This module focuses on the fundamentals of algorithmic thinking. Before diving into complex syntax, students learn to visualize problems, create logical flows, and solve mathematical equations using Flowcharts and basic Python concepts.

**Key Learning Outcomes:**

  * Understanding the logic of **Algorithms** and **Pseudocode**.
  * Mastering **Flowcharts** using Flowgorithm.
  * Learning **Variables, Data Types, and Operators**.
  * Implementing **Loops (For, While)** and **Conditional Statements**.
  * Using **Functions** and **Arrays** for mathematical problem solving.
  * Introduction to **Turtle Graphics** for visual mathematics.
